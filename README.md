
# 🎨 My Art Gallery

A simple, responsive web application to showcase my artworks. Built with **HTML**, **CSS**, and **JavaScript**, and hosted on **GitHub Pages**.

---

## 📌 Features
- Dynamic gallery powered by `artworks.json`
- Filter artworks by **Year** and **Category**
- Lightbox for full-size image viewing
- Captions and descriptions for each artwork
- Easy to update: just edit `artworks.json` and add images
